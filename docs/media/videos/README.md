# SketchForge Demo Video Shot List

Record short clips for the main README. Aim for 10-30 seconds per clip.

The first clip is already included:

```text
01-create-and-edit-block.mp4
01-create-and-edit-block-preview.gif
```

Use these filenames:

```text
01-create-and-edit-block.mp4
02-import-stl.mp4
03-hole-grouping.mp4
04-rotated-hole-cut.mp4
05-export-selected-vs-all.mp4

```

## 01-create-and-edit-block.mp4

Show:

1. Create a new project.
2. Add a Box.
3. Resize it.
4. Lift it.
5. Rotate it.
6. Return to dashboard and show the thumbnail.

## 02-import-stl.mp4

Show:

1. Import an STL from the dashboard.
2. Open it in the editor.
3. Move or scale it.
4. Add a primitive next to it.

## 03-hole-grouping.mp4

Show:

1. Add a solid Box.
2. Add a cutter shape.
3. Switch cutter shape to Hole.
4. Select both shapes.
5. Group them.
6. Orbit to show the cut.

## 04-rotated-hole-cut.mp4

Show:

1. Add a solid.
2. Add a hole.
3. Rotate the hole.
4. Group the result.
5. Orbit around the cut.

## 05-export-selected-vs-all.mp4

Show:

1. Create three shapes.
2. Select two shapes.
3. Export STL.
4. Deselect everything.
5. Export STL again.

## 06-dashboard-thumbnails.mp4

Show:

1. Create a project.
2. Edit the scene.
3. Go home.
4. Show the thumbnail update.
5. Delete a project.
6. Confirm deletion.
